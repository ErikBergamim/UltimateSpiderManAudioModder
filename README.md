# UltimateSpiderManAudioModder
.NET program that automates the conversion of WAV files for insertion into the Ultimate Spider-Man 2005 game, made for all platforms.

# Credits
-Tool for converting to PC format (custom version): https://github.com/FFmpeg/FFmpeg
-Tool for converting to Xbox format: https://github.com/Sergeanur/XboxADPCM
-Tool for converting to PS2 format: https://gamebanana.com/tools/6656
-Tool for converting to Gamecube format: https://github.com/jackoalan/gc-dspadpcm-encode
