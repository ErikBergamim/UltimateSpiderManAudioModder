# UltimateSpiderManAudioModder

.NET program that automates the conversion of WAV files for insertion into the Ultimate Spider-Man 2005 game, made for all platforms.

## Credits

- Tool for converting to PC format (custom version): [FFmpeg](https://github.com/FFmpeg/FFmpeg)
- Tool for converting to Xbox format: [XboxADPCM](https://github.com/Sergeanur/XboxADPCM)
- Tool for converting to PS2 format: [PS2 Tool](https://gamebanana.com/tools/6656)
- Tool for converting to Gamecube format: [gc-dspadpcm-encode](https://github.com/jackoalan/gc-dspadpcm-encode)
